**Not-Humans-Lab** is a thin root workspace that federates four independent AI-infrastructure sub-projects (SkillStack, Deep-Research, Agent-Hub, rate-limit-observatory) — each retaining its own toolchain rather than a single shared build graph.

It is polyglot by design: a **Turborepo + pnpm** workspace with Next.js apps, Python services on **uv + Temporal** (durable workflow orchestration) + FastAPI, Go services, and a **Qdrant** vector store. Root tooling standardizes commit hygiene (Commitizen, commitlint, Husky) across the heterogeneous sub-projects.

- Federates 4 AI-infra sub-projects
- Temporal durable workflows · Qdrant vectors
- Polyglot: TypeScript · Python · Go
